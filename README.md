# 🚀 Tica E-Ticaret Sistemi

Laravel + MySQL altyapısı ile geliştirilmiş, modüler mimariye sahip, gelişmiş SEO, kampanya ve raporlama özellikleri içeren profesyonel e-ticaret sistemidir.

---

## ⚙️ Altyapı

- Laravel
- MySQL
- TailwindCSS
- Modüler mimari
- Trait tabanlı backend yapısı
- Responsive arayüz

---

## 💳 Ödeme Sistemleri

- Sanal POS entegrasyonu
- Havale / EFT ödeme yöntemi
- Çoklu ödeme altyapısı

---

## 🔎 SEO & Reklam Entegrasyonu

- Gelişmiş meta yönetimi
- Google Ads / Analytics / Pixel script alanı
- Panelden script key girme
- Sayfa bazlı head script tetikleme
- Sepet / ürün / dönüşüm takip kodları

---

## 🛍️ Ürün Sistemi

- İç içe kategoriler
- Nitelik / terim tabanlı ürünler
- Terimlere görsel yükleme (max 200kb)
- Stok takibi
- Vergi hesaplama
- Varyasyon mantığı

Örnek:
Ahşap / Renk / Kumaş / Ölçü / Model vb.

---

## 🎟️ İndirim Sistemi

- Kupon kodu
- Herkese açık indirim
- Kategori indirimi
- Ürün indirimi
- Sepet indirimi
- Kampanya kuralları

---

## 🧩 Modüler Ana Sayfa

- Dinamik içerik blokları

## 📦 Excel Toplu Ürün Yönetimi

- Yükleme geçmişi
- Geri yükleme noktası
